# 📊 Project Risk & Health Dashboard

![Version](https://img.shields.io/badge/version-1.0.0-blue)
![Platform](https://img.shields.io/badge/platform-Web-green)
![Status](https://img.shields.io/badge/status-Mockup_Ready-success)

Aplikasi *dashboard* berbasis web interaktif untuk memonitor profil risiko dan kesehatan sebuah proyek (*Project Risk & Health Dashboard*). Proyek ini dirancang sebagai tampilan *mockup* untuk visualisasi data secara elegan, informatif, dan dinamis langsung di *browser* Anda.

---

## ✨ Fitur Utama

- **📈 Visualisasi Dinamis:** Menampilkan skor kesehatan finansial, progres eksekusi, kesiapan suplai, hingga profil risiko menggunakan visualisasi grafik dan warna standar industri (Gauge, S-Curve, Bar Charts).
- **🗺️ Risk Profile Heat Map:** Pemetaan risiko komprehensif pada matriks 5x5 interaktif berdasarkan *Likelihood* (kemungkinan) dan *Consequence* (dampak) dari suatu risiko.
- **⚡ Client-Side Excel Import:** Anda dapat memperbarui data *dashboard* ini secara instan langsung dari *browser* dengan cara memuat file Excel lokal tanpa perlu mengunggahnya ke server. Data Anda sepenuhnya aman.
- **🖨️ Export Ready:** Mendukung pengunduhan *dashboard* secara langsung dalam format gambar resolusi tinggi (PNG/JPG) atau dokumen PDF berukuran A4 yang siap untuk dilaporkan.
- **💻 Responsive Design:** Antarmuka modern bergaya *clean/glassmorphism* dengan perpaduan palet warna indikator performa (Hijau, Kuning, Oranye, Merah) yang nyaman di mata.
